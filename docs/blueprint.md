# **App Name**: TrackBooker

## Core Features:

- User Authentication & Access Control: Allows user registration, login, and manages role-based access for passengers and administrators.
- Train Search & Live Availability: Enables users to search for trains based on source, destination, and date, displaying real-time seat availability.
- Secure Ticket Booking Workflow: Facilitates the process of selecting a train, entering passenger details, confirming the booking, and generating a unique Booking ID (PNR).
- Admin Train & Schedule Management: Provides administrators with tools to add, edit, delete trains, manage routes, schedules, and update seat availability.
- Personalized Booking History & Cancellation: Users can view their past and upcoming bookings, access booking details, and initiate ticket cancellations.
- Integrated Payment Status Tracking: Records and displays the payment status (success, pending, failed) for each booking within the system.
- Admin System Overview & Analytics: A dashboard providing administrators with a high-level view of user, train, and booking statistics using aggregated data.

## Style Guidelines:

- Primary color: Deep sky blue (#2866A3) for headings and interactive elements. Background: Very light desaturated blue-gray (#EFF4F7) for a clean, airy feel. Accent: Vibrant orange-gold (#F5A333) for call-to-action buttons and highlights.
- Body and headline font: 'Inter' (sans-serif) chosen for its modern, neutral, and highly readable characteristics, suitable for conveying information clearly and efficiently across the application.
- Utilize simple, clean, and modern outline icons that integrate seamlessly with the minimal aesthetic. QR codes for tickets will be designed for high contrast and scannability.
- A fully responsive design ensuring optimal viewing and interaction across all device sizes (mobile and desktop). The interface will leverage a card-based layout for organized information display, complemented by a clean, intuitive navigation system featuring both a Navbar and a Sidebar for easy access. This includes consideration for a dark mode toggle.
- Subtle loading animations will provide feedback during data retrieval or submission. Concise toast notifications will deliver real-time system messages and alerts, ensuring users are informed of actions and status updates.
